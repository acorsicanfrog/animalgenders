# Animal Genders

Adds a male and female gender system to animal entities: https://modrinth.com/project/animal-genders

## Overview
This mod introduces gender mechanics to animals, affecting breeding and certain gender-specific abilities. It enhances immersion while staying lightweight and compatible with most mods.

## Features
- Animals are assigned either male or female
- Breeding now requires a male + female pair
- Gender is visually and mechanically recognized

## Female-Exclusive Abilities
- Chickens can lay eggs
- Cows/goats can be milked

## Configuration
The mod includes a configurable file allowing you to:
- Whitelist or blacklist specific entities from having genders
- Adjust the male/female ratio at gender assignment (Applies to naturally generated adults; does not affect newborns)

## Compatibility
Designed for broad compatibility with other animal-related mods.
The mod applies to entities extending the Animal class, ensuring it works with most modded animals automatically.
